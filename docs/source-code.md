Anyways, let's start!

First, let's install dependencies on both <a name="client">**Client** </a> and <a name="server">**Server**.</a>

## [Client](#client)

First, let's open the python script that will setup our dependencies.

Go to `.../src/common` and in the file path type `cmd`

Then, a Command Prompt window will normally open. Type `python setup.py` OR if you compiled an exe, double click it.

A window should open asking you to select either client or server. Type 'client'. Follow the prompts if it asks for any. <br>
After some time, it should have installed. Close the window once finished.

After, navigate to `/src/client` and type `cmd` again in the filepath and type in `client_app.py` OR if you compiled the exe, double click it. <br>
A window should appear. If it says that you didn't install dependencies, please do so!

But if it didn't then your good to go 👍

Select your language, enter the server IP address when prompted, your username and then password. <br> 

Select "Access my files" and you are good to go! Everything should be encrypted but it is recommended to only use local and not online.
